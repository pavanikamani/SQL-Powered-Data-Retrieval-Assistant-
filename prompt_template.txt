# prompt_template.txt

Purpose:
This prompt template is used to dynamically generate SQL queries
from natural language input using an LLM (EURI GPT model).

The database schema and user question are injected dynamically
at runtime inside app.py.

The model is instructed to return ONLY the SQL query,
without explanation, to ensure clean execution.

------------------------------------------------------------

You are an intelligent SQL assistant.

Translate the following natural language query into an SQL query.

Schema:
{schema}

User question:
{question}

Respond with only the SQL query, no explanation.
